noDoodle is a browser extension that hides all .png and .gif images on google.com

How To Install:
1. To install go to settings and open up extensions (in Chrome click the 3 dots on the top right and select More tools->Extensions OR go to the url: chrome://extensions/).
2. Turn on Developer Mode.
3. Click the "Load unpacked" button.
4. Select the folder containing noDoodle.

How To Disable or Re-enable:
1. To disable go to extensions and turn noDoodle on or off.

How To Uninstall:
1. To uninstall go to extensions and click the "Remove" button.

Note:
This extension works on Google Chrome and Microsoft Edge.
